## Description
This project is a simple Flask-based authentication system developed using Python.  
It allows users to register, log in, and log out securely.  
The project is created for learning backend development and authentication concepts.

## Features
- User Registration
- User Login
- User Logout
- Password Hashing
- Session Management
- Basic Input Validation

## Tech Stack
- Python
- Flask
- SQLite / MySQL
- HTML, CSS

## How to Run the Project
1. Clone the repository  
   git clone https://github.com/Sak12345641/flask-authentication-system/tree/main?tab=readme-ov-file

2. Go to the project folder  
   cd flask-authentication-system

3. Install dependencies  
   pip install flask

4. Run the application  
   python app.py

5. Open browser and visit  
   http://127.0.0.1:5000/
